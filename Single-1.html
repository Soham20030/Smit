RAJKOT MUNICIPAL CORPORATION (RMC) CHATBOT USING DIALOGFLOW

The Rajkot Municipal Corporation (RMC) Chatbot is an innovative solution designed to streamline user interactions with municipal services. By harnessing the power of Dialogflow's Natural Language Processing (NLP) and
Intent Classification, the chatbot offers efficient and intuitive communication, allowing users to seamlessly query information and access services. Powered by FastAPI Python backend, the chatbot ensures smooth functionality,
enabling rapid response times and robust performance. The frontend, meticulously crafted using HTML/CSS, provides users with an intuitive and visually appealing interface, enhancing the overall user experience. With its
integration of cutting-edge technologies and a user-centric design approach, the RMC Chatbot represents a significant step forward in enhancing accessibility and efficiency in municipal service delivery. This project is
hosted on GitHub, ensuring transparency and facilitating collaboration for further enhancements and developments.
